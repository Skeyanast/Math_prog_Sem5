## Практическое задание

По заданию было необходимо на основании шифра Виженера (шифрование с помощью ключа) реализовать калькулятор-шифратор-дешифратор информации.

Приложение реализовано на WinForms на языке программирования C#, использован паттерн MVVM.

### Окно программы

Окно программы содержит:

1. Текстовое поле, которое содержит текст для обработки.
2. Текстовое поле, которое содержит ключ.
3. Текстовое поле, которое содержит результат обработки.
4. Кнопка для запуска активной операции.
5. Кнопки типа радио для включения режимов обработки.
